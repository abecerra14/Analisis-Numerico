01 · Introducción

¿Qué es la interpolación polinómica?

Cuando solo tienes puntos discretos y necesitas la función continua que los une, la interpolación polinómica es tu respuesta.

🎯 El problema

En ingeniería de sistemas es común tener datos muestrales —mediciones de sensores, registros de bases de datos, valores tabulados— y necesitar estimar valores intermedios que no fueron medidos.

¿Cómo estimar f(x*) dado {(x₀,y₀), ..., (xₙ,yₙ)}?

✅ La solución

La interpolación polinómica garantiza la existencia y unicidad de un polinomio P(x) de grado ≤ n que pasa exactamente por todos los n+1 puntos dados.

P(xᵢ) = yᵢ para todo i = 0, 1, ..., n

02 · Marco Teórico

Fundamentos Matemáticos

La base rigurosa que sustenta los tres métodos de construcción del polinomio interpolante.

📐 Definición Formal

Dado un conjunto de n+1 puntos distintos {(x₀,y₀), (x₁,y₁), ..., (xₙ,yₙ)} con xᵢ ≠ xⱼ para i ≠ j, el problema consiste en encontrar un polinomio P(x) de grado a lo sumo n tal que:

P(xᵢ) = yᵢ, para todo i = 0, 1, 2, ..., n

🔒 Teorema de Existencia y Unicidad

Teorema: Dados n+1 puntos con abscisas distintas, existe un único polinomio P(x) de grado ≤ n que interpola dichos puntos.

Demostración de unicidad: Si P(x) y Q(x), ambos de grado ≤ n, interpolan los mismos n+1 puntos, entonces D(x) = P(x) − Q(x) tiene grado ≤ n y posee n+1 raíces. Esto solo es posible si D(x) ≡ 0, es decir, P(x) = Q(x). □

⚠️ Implicación práctica: Los tres métodos (Vandermonde, Lagrange, Newton) producen exactamente el mismo polinomio. Las diferencias están en la eficiencia computacional, no en el resultado.

📏 Error de Interpolación

Si f(x) es n+1 veces derivable en [a,b], el error de interpolación está acotado por:

E(x) = f⁽ⁿ⁺¹⁾(ξ) / (n+1)! · (x−x₀)(x−x₁)···(x−xₙ)

El error depende de: la derivada (n+1)-ésima de la función, y la distancia del punto evaluado a cada nodo.

03 · Los Tres Métodos

Vandermonde · Lagrange · Newton

Tres formas de construir el mismo polinomio único, con ventajas y desventajas distintas.

Método de Vandermonde

El enfoque más directo: escribir el polinomio en forma canónica e imponer las condiciones de interpolación para obtener un sistema lineal.

P(x) = a₀ + a₁x + a₂x² + ··· + aₙxⁿ
V · a = y
O(n³) complejidad Baja estabilidad No incremental
La matriz de Vandermonde puede estar mal condicionada para n grande, causando errores numéricos significativos.
import numpy as np def interp_vandermonde(x, y, x_eval): n = len(x) # Construye la matriz de Vandermonde V = np.vander(x, increasing=True) # Resuelve el sistema V·a = y coef = np.linalg.solve(V, y) # Evalúa el polinomio return sum(coef[i] * x_eval**i for i in range(n))

Estructura de la Matriz de Vandermonde (n=4)

1 x₀ x₀² x₀³
1 x₁ x₁² x₁³
1 x₂ x₂² x₂³
1 x₃ x₃² x₃³

Método de Lagrange

Construye el polinomio directamente como combinación lineal de polinomios base Lᵢ(x), sin necesidad de resolver un sistema lineal.

P(x) = Σ yᵢ · Lᵢ(x)
Lᵢ(x) = Π (x−xⱼ)/(xᵢ−xⱼ), j≠i

Cada Lᵢ(x) vale 1 en el nodo xᵢ y 0 en los demás nodos.

O(n²) complejidad Estabilidad media No incremental
Al agregar un nuevo punto, hay que recalcular todos los polinomios Lᵢ(x) desde cero.
def interp_lagrange(x, y, x_eval): n = len(x) resultado = 0.0 for i in range(n): L = 1.0 for j in range(n): if i != j: L *= ((x_eval - x[j]) / (x[i] - x[j])) resultado += y[i] * L return resultado

Propiedad de los polinomios base

Lᵢ(xᵢ) = 1
En su propio nodo
Lᵢ(xⱼ) = 0
En los demás nodos

Método de Newton (Diferencias Divididas)

Construye el polinomio de manera incremental usando diferencias divididas. Al agregar un nuevo punto, solo se calcula un nuevo término.

P(x) = f[x₀] + f[x₀,x₁](x−x₀) + f[x₀,x₁,x₂](x−x₀)(x−x₁) + ···

Las diferencias divididas se calculan recursivamente:
f[xᵢ,...,xᵢ₊ₖ] = (f[xᵢ₊₁,...,xᵢ₊ₖ] − f[xᵢ,...,xᵢ₊ₖ₋₁]) / (xᵢ₊ₖ − xᵢ)

O(n²) complejidad Alta estabilidad ✓ Incremental
⭐ Recomendado: Al agregar un nuevo punto, solo se calcula un nuevo término. Ideal cuando los datos llegan progresivamente.
def dif_divididas(x, y): n = len(x) tabla = np.copy(y).astype(float) for j in range(1, n): for i in range(n-1, j-1, -1): tabla[i] = ((tabla[i] - tabla[i-1]) / (x[i] - x[i-j])) return tabla def interp_newton(x, y, x_eval): coef = dif_divididas(x, y) n = len(x) resultado = coef[n-1] # Evaluación eficiente (Horner) for i in range(n-2, -1, -1): resultado = (resultado * (x_eval - x[i]) + coef[i]) return resultado

06 · Fenómeno de Runge

El Peligro del Grado Alto ⚡

Más nodos no siempre significa mejor aproximación. Para nodos equiespaciados, el polinomio puede oscilar violentamente.

🎯 Función clásica de Runge: f(x) = 1 / (1 + 25x²) en [−1, 1]
Con nodos equiespaciados, el polinomio exhibe oscilaciones crecientes cerca de los extremos del intervalo.
7

❌ Problema

Con nodos equiespaciados y grados elevados (n > 10), el error en los extremos del intervalo crece de forma descontrolada.

✅ Soluciones

🔵 Nodos de Chebyshev: redistribuyen puntos hacia los extremos
🟢 Splines cúbicos: polinomios de bajo grado por tramos, estables y suaves

08 · ¿Por qué este método?

Interpolación vs. Otros Métodos Numéricos

La interpolación polinómica resuelve un tipo de problema diferente a los otros métodos del curso. Entender cuándo usar cada uno es la habilidad clave del ingeniero.

💡 Principio fundamental: No existe "el mejor método" en general. Existe el método correcto para cada tipo de problema. La interpolación no compite con bisección ni con Gauss — resuelven preguntas distintas.
📐 Método de Bisección Búsqueda de raíces ▼

¿Qué resuelve?

Encontrar x donde f(x) = 0. Divide el intervalo [a,b] a la mitad repetidamente hasta converger a la raíz.

¿Cuándo usar bisección?

Cuando necesitas encontrar el cero de una función continua. Garantiza convergencia si f(a)·f(b) < 0.

¿Por qué interpolación aquí?

En el proyecto, el problema es reconstruir f(x) a partir de datos discretos, no encontrar sus raíces. Son preguntas distintas.

🔁 Método de Newton-Raphson Búsqueda de raíces ▼

¿Qué resuelve?

Raíces de ecuaciones no lineales usando la tangente. Convergencia cuadrática muy rápida cerca de la raíz.

Ventajas

Convergencia muy rápida (cuadrática). Pero requiere conocer f'(x), puede divergir si la inicial es mala.

Diferencia clave

Newton-Raphson necesita una expresión analítica de f(x). La interpolación funciona con solo los datos, sin conocer la función.

📏 Método de la Secante Búsqueda de raíces ▼

¿Qué resuelve?

Raíces de ecuaciones, como Newton-Raphson pero aproximando la derivada con una secante entre dos puntos previos.

Ventaja sobre Newton-Raphson

No requiere calcular f'(x) explícitamente. Convergencia superlineal (≈1.618, número áureo).

vs. Interpolación

Mismo argumento: resuelve raíces, no reconstrucción. La interpolación no busca ceros, sino la función que pasa por todos los datos.

🔢 Eliminación de Gauss Sistemas lineales ▼

¿Qué resuelve?

Sistemas de ecuaciones lineales Ax = b, transformando la matriz en forma escalonada para luego hacer sustitución hacia atrás.

Relación con interpolación

El método de Vandermonde usa Gauss para resolver V·a = y. Son complementarios: Gauss es una herramienta, Vandermonde es el método de interpolación.

¿Por qué no solo Gauss?

Gauss resuelve el sistema, pero Newton y Lagrange evitan plantear el sistema completo, siendo más eficientes (O(n²) vs O(n³)).

🔄 Jacobi y Gauss-Seidel Métodos iterativos ▼

¿Qué resuelven?

Sistemas lineales grandes mediante iteración. Parten de una estimación inicial y refinan progresivamente la solución.

Ventaja principal

Para matrices grandes y dispersas, los métodos iterativos son más eficientes en memoria que Gauss directo. Gauss-Seidel converge más rápido que Jacobi.

vs. Interpolación

Misma distinción: solucionan sistemas lineales. La interpolación no busca solucionar un sistema (salvo Vandermonde), sino reconstruir una función continua.

09 · Autoevaluación

¿Cuánto aprendiste? 🎯

Pon a prueba tu comprensión con estas preguntas de repaso.

1. ¿Cuántos polinomios de grado ≤ n existen que interpolen exactamente n+1 puntos distintos?

✅ Correcto. El Teorema de Existencia y Unicidad garantiza que existe exactamente un polinomio de grado ≤ n que pasa por n+1 puntos con abscisas distintas.

2. ¿Qué ventaja tiene el método de Newton sobre Lagrange para datos que llegan progresivamente?

✅ Correcto. Newton es incremental: solo agrega un término nuevo sin recalcular los anteriores. Lagrange debe recomputar todos los Lᵢ(x).

3. El fenómeno de Runge ocurre cuando...

✅ Correcto. Con nodos equiespaciados y grados elevados, el polinomio interpolante puede oscilar violentamente cerca de los extremos del intervalo, aunque pase exactamente por todos los puntos.

4. Si quieres resolver el sistema Ax = b, ¿cuál método usas?

✅ Correcto. Gauss (directo) y Jacobi/Gauss-Seidel (iterativos) son los métodos para sistemas lineales. La interpolación resuelve otro problema: reconstruir una función.
10 · Conclusiones

Síntesis y Recomendaciones

Qué llevarse de este estudio.

1

La interpolación polinómica garantiza la existencia y unicidad de un polinomio de grado ≤ n que pasa exactamente por n+1 puntos dados.

2

Los métodos de Vandermonde, Lagrange y Newton producen el mismo polinomio, pero difieren significativamente en eficiencia computacional, estabilidad numérica y flexibilidad.

3

El método de Newton es la alternativa más recomendable: naturaleza incremental, complejidad O(n²) y alta estabilidad numérica.

4

El fenómeno de Runge demuestra que más nodos equiespaciados no siempre mejoran la aproximación: pueden provocar oscilaciones crecientes.

5

En aplicaciones reales, los splines cúbicos suelen ser preferibles: ofrecen estabilidad sin sacrificar continuidad, independientemente del número de nodos.

6

Cada familia de métodos numéricos resuelve un tipo diferente de problema: raíces, sistemas lineales o reconstrucción de funciones. El ingeniero debe identificar correctamente el problema antes de elegir el método.
